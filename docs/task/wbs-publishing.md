## 퍼블 검토 체크리스트

### HTML 마크업 검토
- [ ] 시맨틱 마크업이 적절하게 사용되었는가?
- [ ] ARIA 라벨이 필요한 곳에 적용되었는가?
- [ ] 폼 요소에 label이 연결되었는가?
- [ ] 이미지에 alt 텍스트가 포함되었는가?
- [ ] 링크, 버튼 등 인터랙션 요소가 명확한가?

### CSS 스타일 검토
- [ ] 디자인 시안과 일치하는가?
- [ ] 컬러, 폰트, 간격 등이 일관되게 적용되었는가?
- [ ] 반응형 웱이 제대로 구현되었는가?
- [ ] 다국어 텍스트 길이 대응이 되어 있는가?
- [ ] CSS 클래스 명명 규칙을 따르는가?

### 반응형 웹 검토
- [ ] 데스크톱/태블릿/모바일 화면이 최적화되었는가?
- [ ] 화면 회전 시 대응이 되어 있는가?
- [ ] 터치 기반 인터랙션이 구현되었는가?
- [ ] 미디어 쿼리가 적절하게 설정되었는가?
- [ ] 텍스트 크기, 버튼 크기 등이 모바일에 적절한가?

### 다국어 지원 검토
- [ ] 한국어/영어 텍스트가 모두 정상 표시되는가?
- [ ] 텍스트 길이 차이에 따른 레이아웃 조정이 되어 있는가?
- [ ] 언어 선택 기능이 구현되었는가?

### 접근성 검토
- [ ] WCAG 색상 대비를 준수하는가?
- [ ] 키보드 네비게이션이 가능한가?
- [ ] 스크린 리더 지원이 되는가?
- [ ] 포커스 상태가 명확히 표시되는가?

### 크로스 브라우저 호환성 검토
- [ ] Chrome, Firefox, Safari, Edge에서 정상 작동하는가?
- [ ] 각 브라우저별 CSS 접두사가 필요한 곳에 적용되었는가?
- [ ] 각 브라우저별 렌더링 차이가 최소화되었는가?
