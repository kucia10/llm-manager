# 기획 파트별 WBS (Work Breakdown Structure)

## 역할 개요

- **담당자:** 기획자 (1명)
- **기간:** 2026년 1월 ~ 2026년 2월 (2개월)
- **총 M/M:** 2.0 M/M
- **주요 책임:** 요구사항 분석, 기획서 작성, 프로토타입 기획, 디자인 지원

---

## 기획 담당 작업 일정 (전체 기간)

| Phase | 기간 | 주요 작업 | 소요 M/M |
|-------|------|----------|----------|
| Phase 1 | 1월 1주 | 프로젝트 이해 및 초기 분석 | 0.1 |
| Phase 2 | 1월 2~4주 | 요구사항 분석 및 기획서 작성 | 1.4 |
| Phase 3 | 2월 1~2주 | 설계 검토 및 기획 수정 | 0.2 |
| Phase 4 | 2월 1~4주 | 디자인 기획 지원 | 0.3 |
| **합계** | **전체 기간** | | **2.0** |
